## 9. Entry Structure Anomalies

| Issue | Description |
|:---|---|
| Caption-only parsing miss | 62+ entries had NO text captions -- only document filenames. Initial tree (105 entries) only included entries with captions. Full rebuild from filenames found ~238 raw entries. |
| Filename bracket inconsistency | 1,345+ unique bracket patterns for season/episode info ([S1 EP01], [S01EP01], [S1-EP01], [S1 E01], etc.) -- made automated season parsing unreliable for detailed episode breakdown. |
| Mixed alias usage in filenames | Some uploaders used English titles ("Reincarnated as a Slime"), others used Japanese abbreviations ("Tensura"), causing the same anime to appear under different keys. |
| Filename vs caption title mismatch | Caption text and document filenames sometimes used different titles/names for the same anime, requiring cross-referencing. |
| Resolution treated as entry name | Entries like "Blue Exorcist: The Movie 1080p" had resolution baked into the title -- required regex stripping to merge. |
| Spin-off detection | "Slime Diaries" and "Gun Gale Online" are spin-offs of larger franchises -- required manual identification to merge correctly. |
